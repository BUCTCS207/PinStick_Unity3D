注：脚本是用C#写的（在Unity3D中脚本可以当作组件使用，进行拖拽）


1 .unitypackage是存放素材的（可以直接拖拽至Unity3D中，此处不涉及到，以后 可能用到）


2. UI设计（或称界面设计）是指对软件的人机交互、操作逻辑、界面美观的整体设计。UI设计分为实体UI和虚拟UI，互联网常用的UI设计是虚拟UI，UI即User Interface(用户界面)的简称。
好的UI设计不仅是让软件变得有个性有品位，还要让软件的操作变得舒适简单、自由，充分体现软件的定位和特点。


3. 创建UI 
（1）Text UI：用于记录分数
	1): Rect Transform: 用于调整Text的大小
	2): Text(Scripts): 里面可以设置字体样式，大小，居中等
（2）画布Canvas：是Text UI 所在的画布（创建的UI都会在里面 ？）


4. 创建旋转脚本
（1）Rotation.cs中有一个需要注意的地方：new Vector3(0, 0, speed * Time.deltaTime)，解释：unity的生命周期函数有两种1. Update()，2. FixedUpdate()；第一种是按帧率刷新，第二种是按固定时间
进行刷新，由于机器不同（甚至相同机器的美每帧的时间都不同），因此speed * Time.deltaTime保证每秒旋转的角度为90度


5.  创建针头
（1）创建：就是长方形头上放一个圆
（2）设为预制体：因为要用很多针，所以将其作为预制体，方便实例化
（3）为针头添加碰撞检测（针身不必添加）（若添加对象是预制体，则用Apply应用到所有预制体）

6. 判断游戏结束的条件：游戏结束是根据针头的碰撞进行判断的，所以应该给针头添加刚体属性（用于碰撞检测）
